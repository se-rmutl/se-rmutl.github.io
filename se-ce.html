วิศวกรรมซอฟต์แวร์ (SE) vs วิศวกรรมคอมพิวเตอร์ (CoE)

สองสาขายอดฮิตในโลกเทคโนโลยีที่ดูคล้ายแต่แตกต่างกัน มาดูกันว่าสาขาไหนที่เหมาะกับคุณ

วิศวกรรมซอฟต์แวร์

หัวใจหลัก

เน้นกระบวนการสร้างและพัฒนา ซอฟต์แวร์ขนาดใหญ่ (Large-scale Software) อย่างเป็นระบบ ตั้งแต่การเก็บความต้องการ, ออกแบบ, พัฒนา, ทดสอบ, ไปจนถึงการบำรุงรักษา

เรียนอะไรบ้าง?

  • ◆กระบวนการพัฒนาซอฟต์แวร์ (Agile, Scrum)
  • ◆การออกแบบสถาปัตยกรรมซอฟต์แวร์
  • ◆การประกันคุณภาพซอฟต์แวร์ (QA)
  • ◆การจัดการโครงการซอฟต์แวร์
  • ◆การพัฒนาเว็บและแอปพลิเคชัน

อาชีพในอนาคต

Software Developer Web/Mobile Developer DevOps Engineer QA / Test Engineer Project Manager

วิศวกรรมคอมพิวเตอร์

หัวใจหลัก

เน้นการออกแบบและสร้างระบบคอมพิวเตอร์ทั้ง ฮาร์ดแวร์ (Hardware) และ ซอฟต์แวร์ (Software) ระดับล่างที่ควบคุมฮาร์ดแวร์โดยตรง เป็นการเชื่อมโลกกายภาพเข้ากับโลกดิจิทัล

เรียนอะไรบ้าง?

  • ◆วงจรไฟฟ้าและอิเล็กทรอนิกส์
  • ◆สถาปัตยกรรมคอมพิวเตอร์
  • ◆ระบบปฏิบัติการ (Operating Systems)
  • ◆เครือข่ายคอมพิวเตอร์ (Networks)
  • ◆ระบบสมองกลฝังตัว (Embedded Systems)

อาชีพในอนาคต

Embedded Systems Engineer Hardware Engineer Network Engineer Firmware Developer System Engineer

เปรียบเทียบเชิงลึก

วิศวกรรมซอฟต์แวร์

วิศวกรรมคอมพิวเตอร์

รายวิชาที่เน้น

เน้นการจัดการและกระบวนการ เช่น การวิเคราะห์ความต้องการ, การออกแบบสถาปัตยกรรมซอฟต์แวร์, การทดสอบระบบ, การบริหารโครงการ

รายวิชาที่เน้น

เน้นหลักการพื้นฐานของคอมพิวเตอร์ เช่น วงจรดิจิทัล, สถาปัตยกรรมคอมพิวเตอร์, ระบบปฏิบัติการ, การสื่อสารข้อมูลและเครือข่าย

พื้นฐานที่ควรมี

ทักษะการคิดเชิงตรรกะ, การแก้ปัญหาอย่างเป็นระบบ, การสื่อสารและทำงานเป็นทีม, สนใจสร้างแอปพลิเคชันเพื่อแก้ปัญหาให้ผู้ใช้

พื้นฐานที่ควรมี

พื้นฐานคณิตศาสตร์และฟิสิกส์ที่ดี, สนใจการทำงานของฮาร์ดแวร์และระบบระดับล่าง, ชอบลงมือปฏิบัติและทดลอง

สรุปง่ายๆ ในหนึ่งประโยค

วิศวกรซอฟต์แวร์ สร้าง "ความคิดและจิตวิญญาณ" ที่ทำงานบนคอมพิวเตอร์, ส่วน วิศวกรคอมพิวเตอร์ สร้าง "สมองและร่างกาย" ของคอมพิวเตอร์ให้ทำงานได้

อินโฟกราฟฟิกนี้จัดทำขึ้นเพื่อเป็นแนวทางเบื้องต้นในการตัดสินใจ

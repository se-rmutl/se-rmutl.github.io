# Placeholder Image Plan

เว็บเพจฉบับนี้ใช้ Visual Placeholder แบบ CSS เพื่อคงไฟล์ HTML ให้เปิดใช้งานได้ทันทีโดยไม่ต้องพึ่งไฟล์ภาพภายนอก

## ภาพที่ควรสร้างในรอบถัดไป

| Project | ภาพที่ต้องสร้าง | ลักษณะภาพที่แนะนำ |
|---|---|---|
| P1 | Evidence Workbench Dashboard | ภาพหน้าจอ dashboard แบบ evidence list + filters + evidence gap + CQI action |
| P2 | PLO-CLO Mapping Matrix | ตาราง mapping ระหว่างรายวิชา PLO CLO พร้อม I/R/M และ coverage chart |
| P3 | Student Project Journey | Flow Proposal → Review → Revision → Approved → Milestone → Portfolio |
| P4 | Lab Readiness Dashboard | QR asset lookup, room readiness status, maintenance, material stock |
| P5 | Active Learning Journey | Event plan → register → QR check-in → evidence → reflection → engagement dashboard |
| P6 | Stakeholder Insight Dashboard | satisfaction scores, open-ended feedback clusters, issue list to CQI |

## รูปแบบภาพ

- อัตราส่วนแนะนำ: 16:10 หรือ 4:3
- ขนาดอย่างน้อย: 1600 × 1000 px
- ใช้โทน: navy / teal / white และใช้สี accent ประจำโครงการ
- เน้นภาพ mockup หน้าจอระบบหรือ infographic ที่สื่อ workflow โดยไม่ต้องมีข้อมูลจริง

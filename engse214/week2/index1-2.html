Malware Family

Malicious Software (ซอฟต์แวร์ประสงค์ร้าย) มีหลายประเภท พลิกการ์ดดูรายละเอียด:

Virus

VIRUS

"ต้องเกาะไฟล์อื่น"

  • ต้องการ Host File (.exe)
  • ต้องรอคนกดเปิดถึงทำงาน
  • แพร่เชื้อในเครื่องเดียวกัน

Worm

WORM

"ไปเองได้ไม่ง้อคน"

  • Standalone (ไม่ต้องเกาะใคร)
  • แพร่กระจายเองผ่าน Network
  • ใช้ช่องโหว่ระบบ (Exploit)

Trojan

TROJAN

"หน้าเนื้อใจเสือ"

  • ปลอมเป็นโปรแกรมดี
  • ไม่แพร่กระจายเอง
  • เปิดประตูหลัง (Backdoor)

Ransomware

RANSOMWARE

"โจรเรียกค่าไถ่"

  • เข้ารหัสไฟล์ (Encrypt)
  • เรียกเงินแลก Key
  • ห้ามจ่ายเด็ดขาด!

Social Engineering (วิศวกรรมสังคม)

"Hacking the Human"

จุดอ่อนที่สุดของระบบความปลอดภัยไม่ใช่คอมพิวเตอร์ แต่คือ "มนุษย์"

Pretexting

สวมรอยเป็นคนอื่น (เช่น IT Support)

Baiting

ล่อด้วยของฟรี (USB, Free Wi-Fi)

Quid Pro Quo

ยื่นหมูยื่นแมว (ขอรหัสแลกของรางวัล)

Thai Case Study: GSB ATM Jackpotting

เหตุการณ์ (2016)

ธนาคารออมสินถูกโจมตีตู้ ATM ยี่ห้อ NCR

  • คนร้ายใช้ Malware สั่งตู้จ่ายเงินเอง (Jackpotting)
  • ความเสียหาย: เงินสดหายไปกว่า 12 ล้านบาท
  • สาเหตุ: Network ตู้ ATM แยกไม่ขาดจากภายนอก + ใช้ Hard Disk ปกติ
Physical Access + Malware = Disaster

Mobile Security & Fake Apps

Fake Apps (แอปดูดเงิน)

มักมาในรูปของ:

  • แอป "เงินกู้ด่วน" / "ดิจิทัลวอลเล็ต"
  • แอปหาคู่ / ดูบอลฟรี
  • APK นอก Store (Sideloading)

Permission Abuse

สังเกตสิทธิ์ที่ขอเกินความจำเป็น:

  • แอปไฟฉาย ขออ่าน SMS/Contact
  • แอปแต่งรูป ขอสิทธิ์ Accessibility Service (เพื่อควบคุมหน้าจอ)

Thai Case Study: 9near

เหตุการณ์ข้อมูลคนไทย 55 ล้านคนหลุด (2023)

ข้อมูลที่หลุด

สาเหตุ & ผลกระทบ

ช่องโหว่ในระบบของหน่วยงานรัฐ (คาดว่าเกี่ยวกับระบบลงทะเบียนวัคซีน/สวัสดิการ)

Cyber Hygiene: Password

ตารางเวลาที่ Hacker ใช้เจาะรหัสผ่าน (Time to Crack) [Update 2024]

ความยาว ตัวเลขล้วน ตัวเลข+อักษรเล็ก/ใหญ่+สัญลักษณ์
8 ตัว ทันที (Instantly) 5 ชั่วโมง
10 ตัว ทันที 5 วัน
12 ตัว 2 วินาที 200 ปี!
Best Practice: ยาว 12+ ตัว, ไม่ซ้ำกันแต่ละเว็บ, และเปิด 2FA เสมอ

ชวนคิด: Convenience vs Security

Trade-off ที่เราต้องเลือกในชีวิตจริง

ความสะดวก (Convenience)

  • ใช้ Password เดียวทุกเว็บ (จำง่าย)
  • กด "Remember Me" ตลอด
  • ปิด 2FA เพราะขี้เกียจรอ SMS

ผลลัพธ์: แฮกง่ายมาก

ความปลอดภัย (Security)

  • Password ต่างกัน (ต้องใช้ Password Manager)
  • Logout ทุกครั้งหลังใช้
  • ใช้ App Authenticator ยืนยันตัวตน

ผลลัพธ์: ปลอดภัยสูง

APT (Advanced Persistent Threat)

ภัยคุกคามระดับสูง เป้าหมายคือองค์กรใหญ่/รัฐบาล

// APT Characteristics: 1. TARGETED: มีเป้าหมายชัดเจน (ไม่ใช่หว่านแห) 2. PERSISTENT: ฝังตัวยาวนาน (Low and Slow) เป็นเดือน/ปี 3. ADVANCED: ใช้เทคนิคขั้นสูง / Zero-day Exploit 4. STEALTHY: หลบหลีกการตรวจจับของ Antivirus

Supply Chain Attack: SolarWinds

เมื่อ "ยาม" เป็นคนเปิดประตูให้โจร (Software Update ที่ถูกวางยา)

  • แฮกเกอร์เจาะระบบ SolarWinds (ผู้ผลิตซอฟต์แวร์)
  • ฝัง Backdoor ในไฟล์ Update ของโปรแกรม Orion
  • ลูกค้า 18,000 ราย (รวมรัฐบาลสหรัฐฯ) โหลดไปติดตั้งเอง
Lesson: แม้แต่ Software จากแหล่งที่น่าเชื่อถือ ก็อาจไม่ปลอดภัย

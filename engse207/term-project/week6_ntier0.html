Why replace SQLite?

SQLite locks the file during writes. In a real multi-user environment (N-Tier), we need a database engine like PostgreSQL that handles Concurrent Connections.

Why use Nginx? Why not express.static?

Node.js is single-threaded and great for API logic, but bad at serving static files. Nginx is optimized for High Concurrency serving of HTML/CSS/Images and SSL termination.

The Trade-off

Complexity vs. Scalability.
We added 2 new infrastructure pieces (Nginx, Postgres). Setup is harder, but now we can scale each tier independently.

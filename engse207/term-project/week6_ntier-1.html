Why PostgreSQL?

The Problem with SQLite

It's a file. When one user writes, the whole file is locked. It cannot handle concurrent writes well in a multi-tier environment.

The Solution: PostgreSQL

A robust Client-Server database engine. It manages Concurrent Connections, Row-level locking, and handles complex queries efficiently.

Key Commands:

sudo -u postgres psql
CREATE DATABASE taskboard;
CREATE USER admin WITH PASSWORD 'securepass';
2

Enter Nginx

"Don't let users talk to Node.js directly."

  • Security Barrier

    Hide backend port 3000. Prevent direct attacks on app logic.

  • Static Performance

    Nginx serves HTML/CSS/JS 10x faster than Node.js.

/etc/nginx/sites-available/default
location /api/ {
  proxy_pass http://localhost:3000;
  proxy_http_version 1.1;
}

HTTP is NOT SAFE

In N-Tier, data travels across networks.
Without SSL, passwords and data are sent as Plain Text.

Self-Signed Certificate

For this lab, we act as our own Certificate Authority (CA).

openssl req -x509 -nodes -days 365 ...

Browser Warning?

Browsers will warn "Not Secure" because they don't trust our local CA. This is normal for dev/labs.

`let` & `const`

`const` ไม่สามารถกำหนดค่าใหม่ได้ ลองคลิกปุ่มเพื่อดู Error

year = 2026; // TypeError!

MILESTONE 2

การทำงานแบบไม่รอ (Asynchronous)

หัวใจของการสร้างเว็บที่ลื่นไหล คือการจัดการงานที่ใช้เวลานานโดยไม่ทำให้หน้าเว็บค้าง

Promises: "คำสัญญา" แห่งอนาคต

Promise คือ Object ที่เป็นตัวแทนของผลลัพธ์ (สำเร็จหรือล้มเหลว) ที่จะเกิดขึ้นในอนาคต ลองคลิกเพื่อจำลองการทำงานของ Promise

สถานะ: ว่าง

Async/Await Simulation

`await` จะ "หยุดรอ" ให้ Promise ทำงานเสร็จก่อนไปต่อ คลิกเพื่อดูการทำงานทีละขั้น

async function getUser() {
console.log("Fetching user...");
const user = await fakeFetch();
console.log(user);
}

# ENGSE203 Computer Programming for Software Engineer
เว็บไซต์นำเสนอข้อมูลรายวิชา ENGSE203 การเขียนโปรแกรมสำหรับวิศวกรซอฟต์แวร์ มทร.ล้านนา ปีการศึกษา 2569

## วิธีใช้งาน
1. แตกไฟล์ .zip ออก
2. ดับเบิ้ลคลิกเปิดไฟล์ `index.html` บนเว็บเบราว์เซอร์ได้ทันที
3. สามารถนำรูปภาพสไลด์หรือกราฟิกมาวางในโฟลเดอร์ `images/` เพื่ออ้างอิงในเว็บเพจในอนาคต

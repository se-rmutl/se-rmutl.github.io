# ENGSE203 — Computer Programming for Software Engineer (Light Theme Portal)
เอกสารและพอร์ทัลรายวิชา ENGSE203 การเขียนโปรแกรมสำหรับวิศวกรซอฟต์แวร์ มทร.ล้านนา (ปีการศึกษา 2569)

## 📌 สภาพแวดล้อมและเทคโนโลยี
- **Language:** JavaScript (ES6+)
- **Front-End:** React.js, HTML5/CSS3 (Responsive Tailwind)
- **Back-End:** Node.js, Express.js (RESTful API)
- **Database:** SQLite & MongoDB (Mongoose)
- **Testing & Tools:** Jest/Vitest, Git/GitHub, VS Code

## 🚀 วิธีใช้งานเว็บไซต์
1. แตกไฟล์ (.zip) ออกมาที่คอมพิวเตอร์ของคุณ
2. ดับเบิ้ลคลิกเปิดไฟล์ `index.html` บนเว็บเบราว์เซอร์ได้ทันที
3. โฟลเดอร์ `images/` ใช้สำหรับเก็บรูปภาพประกอบ หรือกราฟิกที่ต้องการอ้างอิงบนเว็บเพจในอนาคต

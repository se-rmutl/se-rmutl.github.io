# ENGSE203 — Week 02 Interactive HTML Slides

Interactive single-page teaching slides for **ENGSE203: Computer Programming for Software Engineer** — Week 02.

## Topics
- Modern JavaScript (ES6+)
- ES Modules and separation of responsibilities
- Async/Await, fetch concepts, and error handling
- npm scripts
- Git feature branch / pull request workflow
- GitHub Pages deployment
- LAB 2: ENGSE203 Learning Dashboard

## How to use
1. Extract the ZIP while preserving the `images/` folder.
2. Open `index.html` in a modern browser.
3. For the best projector experience, use the **Full screen** button.
4. Use **Previous / Next**, the slide selector, arrow keys, PageUp/PageDown, or swipe vertically on mobile.
5. Font scale controls persist in the browser until reset.

No external library or Internet connection is required for the interactive examples.

## Notes
- The page is designed at a 16:9 visual ratio on desktop/projector and switches to a vertically responsive layout on narrow screens.
- The images folder contains the course logo, original SVG instructional illustrations, and optional visual references.

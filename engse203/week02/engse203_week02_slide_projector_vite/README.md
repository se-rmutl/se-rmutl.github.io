# ENGSE203 Week 02 — Projector Edition + Vite

สไลด์ HTML แบบ 16:9 สำหรับสอน ENGSE203 สัปดาห์ที่ 2 โดยปรับ typography ให้ใหญ่ขึ้นสำหรับ projector และเพิ่มเนื้อหา Vite เพื่อใช้งานคู่กับ `ENGSE203 Week 02 Interactive Teaching Sandbox`.

## วิธีเปิด

1. แตกไฟล์ ZIP
2. เปิด `engse203_week02_slide_projector_vite/index.html` ด้วย Chrome, Edge หรือ Safari
3. กด `⛶` เพื่อเต็มจอ
4. ใช้ปุ่ม `A− / A+ / A` เพื่อปรับขนาดตัวอักษรระหว่างสอน
5. ใช้ลูกศร `← →`, `Space`, `PageUp/PageDown`, `Home/End` เพื่อเปลี่ยนสไลด์

## เนื้อหาที่เพิ่ม

- บทบาทของ Vite: dev server และ build tool
- เหตุผลที่ไม่ควรทดสอบ ES Modules / fetch จาก `file://`
- HMR และข้อควรระวังเรื่อง state ของ Vanilla JS
- ขั้นตอน source → Vite build → deploy
- คำสั่ง Vite ที่ใช้ใน LAB 2
- ตารางเปรียบเทียบ Vite กับ Pure Vanilla JavaScript
- Interactive Vite Console สำหรับสาธิต `dev`, HMR, `build` และ `base`

## หมายเหตุ

- สไลด์ทำงานแบบ offline ได้ เพราะใช้ CSS/JavaScript แบบ inline และรูปภาพอยู่ใน `images/`
- สไลด์นี้เป็นสื่ออธิบายภาพรวม ให้เปิด Interactive Teaching Sandbox เมื่อต้องการให้นักศึกษาลองโค้ดหรือเปลี่ยนค่าเอง

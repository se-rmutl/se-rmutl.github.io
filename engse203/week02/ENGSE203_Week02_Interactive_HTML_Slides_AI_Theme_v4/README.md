# ENGSE203 Week 02 Interactive HTML Slides — AI-Native Theme v4

เวอร์ชันนี้เป็นอีกธีมสำหรับใช้สอนบน projector โดยคงเนื้อหาจากเอกสาร ENGSE203 สัปดาห์ที่ 2 เดิม แต่ปรับภาพรวมเป็นธีม AI/Neon/Glassmorphism ที่ทันสมัย พร้อม interaction เดิมทั้งหมด

## วิธีใช้งาน

เปิด `index.html` ด้วยเว็บเบราว์เซอร์สมัยใหม่ เช่น Chrome, Edge หรือ Safari

## คุณสมบัติ

- Single HTML Page Slide ขนาด 16:9
- Responsive สำหรับ projector, tablet และ mobile
- ปุ่ม A− / A / A+ สำหรับปรับขนาดฟอนต์
- ปุ่ม Full screen
- ตัวอย่าง JavaScript ที่รันได้จริงใน browser
- Async simulator, npm script selector, Git workflow และ deploy checklist
- โฟลเดอร์ `images/` สำหรับรูปประกอบทั้งหมด

## หมายเหตุ

ไฟล์นี้ออกแบบให้เปิด offline ได้ โดยไม่ต้องเรียก CDN ภายนอก

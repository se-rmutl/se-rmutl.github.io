ENGSE203 Week 02 Single HTML Page Slide

วิธีใช้งาน
1) แตกไฟล์ ZIP
2) เปิด index.html ด้วย Browser หรือเปิดผ่าน local web server ก็ได้
3) ใช้ปุ่ม Prev/Next หรือปุ่มลูกศรซ้าย/ขวาเพื่อเปลี่ยนสไลด์
4) ใช้ปุ่ม A-/A+ เพื่อย่อ/ขยายตัวอักษร
5) ใช้ปุ่ม ⛶ เพื่อเปิดเต็มจอ
6) สไลด์ที่มีปุ่ม Run example / Load success / Simulate error / Explain สามารถทดลอง JavaScript ได้จริง

โครงสร้างไฟล์
- index.html  : single page slide พร้อม CSS/JS ในไฟล์เดียว
- images/     : รูปประกอบทั้งหมดที่ใช้ในสไลด์

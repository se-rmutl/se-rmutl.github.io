# ENGSE203 Week 02 Single HTML Slide — Complete Visual Teaching Edition

ไฟล์นี้เป็นสไลด์ HTML หน้าเดียวขนาด 16:9 สำหรับสอน Week 02 วิชา ENGSE203 ใช้คู่กับ Interactive Teaching Sandbox

## วิธีใช้
- เปิด `index.html` ด้วย Browser
- กด `F` หรือปุ่ม Fullscreen สำหรับฉาย Projector
- ใช้ปุ่ม `A− / A / A+` เพื่อปรับขนาดตัวอักษร
- ใช้ปุ่มลูกศรหรือคีย์บอร์ด `← / →` เพื่อเปลี่ยนสไลด์

## เนื้อหา
ครอบคลุม ES6+, ES Modules, Vite, Async/Await, Error Handling, npm Scripts, Git Workflow, GitHub Pages, LAB 2 Step-by-step, README, Rubric, Checklist, Troubleshooting และ Teaching Record

## โครงสร้าง
- `index.html` — สไลด์ทั้งหมดและ interactive mini demos
- `images/` — รูปประกอบ diagram และ infographic

เริ่มต้น Lab

ภาพรวมของ Lab (The Big Picture)

ใน Lab นี้ เราไม่ได้แค่เขียนโค้ด แต่เรากำลังสร้างรากฐานที่สำคัญของระบบ **"Agent Wallboard"** ซึ่งเป็นเครื่องมือที่ใช้ใน Call Center หรือทีม Support สมัยใหม่ เพื่อติดตามสถานะของพนักงาน (Agent) แบบ Real-time

เป้าหมายหลักของ Phase 1 คือการสร้าง **Backend API** ที่แข็งแกร่ง, ขยายง่าย และมีโครงสร้างแบบมืออาชีพ ซึ่งจะทำหน้าที่เป็น "สมอง" ของระบบทั้งหมด และเป็นทักษะพื้นฐานสำคัญสำหรับตำแหน่ง Backend และ Full-Stack Developer ครับ

สิ่งที่จะได้เรียนรู้ใน Phase 1:
  • ✅ Professional Node.js API Development
  • ✅ MVC Architecture Pattern
  • ✅ Input Validation with Joi
  • ✅ Error Handling & Middleware
  • ✅ API Testing & Documentation

Hour 3: Validation & Error Handling (70 นาที)

สร้าง API ที่ฉลาด ปลอดภัย และจัดการข้อผิดพลาดได้

ทำให้ API ของเราแข็งแกร่งและน่าเชื่อถือโดยการดักจับข้อมูลที่ไม่ถูกต้องและจัดการกับ Error ที่อาจเกิดขึ้นได้อย่างสวยงาม

เขียน Middleware สำหรับ Validation ใน `middleware/validation.js`!

TODO #4: Complete `statusUpdate` Schema (15 นาที)

สร้าง Joi schema สำหรับ validate request body ของการอัปเดตสถานะ โดยมีเงื่อนไข:

  • `status`: ต้องเป็นค่าที่ถูกต้องจาก `AGENT_STATUS` และเป็นค่าที่บังคับ (required).
  • `reason`: เป็น string ที่ไม่บังคับ (optional) และยาวไม่เกิน 200 ตัวอักษร.

รูปแบบการส่งงานมาตรฐาน

  • 1 ส่งเป็น **URL ของ GitHub Repository** หรือ GitHub Classroom ผ่านระบบ LMS/Google Classroom ภายในเวลาที่กำหนด
  • 2 โค้ดใน **main branch ต้องสามารถรันได้จริง** ตามขั้นตอนที่ระบุไว้ในเอกสาร README.md
  • 3 แนบภาพหน้าจอหรือหลักฐานผลการรัน test ตามที่กำหนดในใบงาน
  • 4 งานกลุ่มต้องมีประวัติ commit/issue/branch ที่ตรวจสอบการมีส่วนร่วมของสมาชิกแต่ละคนได้อย่างชัดเจน

Immutable Update

อย่าแก้ Array เดิม—สร้างค่าใหม่

❌ Mutate

ไม่ควรทำ 
requests.push(newRequest);
setRequests(requests);

✓ Immutable

ควรทำ 
setRequests((current) => [
  newRequest,
  ...current,
]);

กดปุ่มเพื่อเปรียบเทียบ reference และ array result

Interactive State

ทดลอง State → Derived Data → Render

App.jsx 
function handleDeleteRequest(requestId) {
  setRequests((current) =>
    current.filter((request) =>
      request.id !== requestId
    )
  );
}

<button onClick={() =>
  handleDeleteRequest(request.id)
}>ลบ</button>

Controlled Form

State เป็น Source of Truth ของทุก Field

RequestForm.jsx 
const [formData, setFormData] = useState({
  requesterName: "",
  requestType: "",
  location: "",
  details: "",
  priority: "normal",
});

function handleChange(event) {
  const { name, value } = event.target;
  setFormData((current) => ({
    ...current,
    [name]: value,
  }));
}
Controlled input 
<label htmlFor="requesterName">
  ชื่อผู้แจ้ง
</label>
<input
  id="requesterName"
  name="requesterName"
  value={formData.requesterName}
  onChange={handleChange}
/>
Controlled pair: ถ้ามี value แต่ไม่มี onChange ผู้ใช้จะพิมพ์ไม่ได้

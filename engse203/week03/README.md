# ENGSE203 Week 03 — Interactive HTML Slides

สื่อการสอนแบบ Single HTML Page Slide ขนาด 16:9 สำหรับสัปดาห์ที่ 3 เรื่อง Web UI, Responsive Layout, Event และ Form

## วิธีใช้งาน

เปิด `index.html` ด้วย browser ได้ทันที หรือใช้ VS Code Live Server เพื่อสอนเต็มจอ

ปุ่มควบคุมในสไลด์:
- ← / → เปลี่ยนสไลด์
- A− / A / A+ ปรับขนาดตัวอักษร
- Full screen เข้าสู่โหมดเต็มจอ
- Dropdown เลือกสไลด์

## โฟลเดอร์

- `index.html` — สไลด์หลักพร้อม JavaScript interactive demos
- `images/` — รูปและ infographic จากเอกสารประกอบการสอน Week 03

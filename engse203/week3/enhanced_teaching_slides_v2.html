🎯 Performance Tips

  • Use transform and opacity - hardware accelerated
  • Avoid animating layout properties (width, height, margin)
  • Use will-change property for complex animations
  • Prefer requestAnimationFrame for JavaScript animations

🚀 Optimization Suggestions

1. Use CSS Variables for Repeated Values

แทนที่ค่าสีที่ใช้ซ้ำด้วย CSS Variables

:root { --primary-color: #3498db; }

2. Combine Similar Selectors

รวม selectors ที่มี properties เหมือนกัน

.btn, .button { ... }

3. Use Shorthand Properties

ใช้ shorthand properties เพื่อลดขนาดไฟล์

margin: 1rem 2rem; /* แทน margin-top: 1rem; margin-right: 2rem; */

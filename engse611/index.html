ปลดล็อกศักยภาพสู่ Full-Stack Developer

เรียนรู้การสร้างเว็บแอปพลิเคชันสมัยใหม่ตั้งแต่ศูนย์จนจบ ครบทั้ง Frontend และ Backend ภายใน 5 วัน!

เรียนแล้วทำอะไรได้บ้าง?

สร้างเว็บไซต์ที่ตอบสนอง

คุณจะสามารถสร้างเว็บไซต์ที่สวยงามและใช้งานได้จริงด้วย `HTML`, `CSS` และ `JavaScript` ตั้งแต่หน้าแนะนำตัวไปจนถึงเว็บที่มีปฏิสัมพันธ์กับผู้ใช้

พัฒนา Interactive App

ยกระดับเว็บของคุณให้เป็นแอปพลิเคชันที่ทันสมัยด้วย `React.js` ซึ่งเป็นเทคโนโลยีที่บริษัทชั้นนำทั่วโลกเลือกใช้

เข้าใจเทคโนโลยี Full-Stack

ไม่ได้เรียนแค่หน้าบ้าน (Frontend) แต่คุณจะได้สร้างหลังบ้าน (Backend) ด้วย `Node.js` และ `Express.js` เพื่อสร้าง `API` ของตัวเอง

สร้างโปรเจคจริงสำหรับ Portfolio

เปลี่ยนความรู้ให้เป็นผลงานที่จับต้องได้! คอร์สนี้จะพาคุณทำโปรเจคจริงที่สามารถนำไปใส่ `Portfolio` เพื่อสมัครงานในอนาคต

เป้าหมายการเรียนรู้ (CLOs)

CLO1

สร้างเว็บไซต์พื้นฐานด้วย `HTML5`, `CSS3` และ `JavaScript` ได้

CLO2

ใช้ `React.js` ในการสร้าง `Interactive Web Application` เบื้องต้นได้

CLO3

สร้างเว็บเซิร์ฟเวอร์ง่ายๆ ด้วย `Node.js` และ `Express.js` ได้

CLO4

เชื่อมต่อ `Frontend` กับ `Backend` และส่งผ่านข้อมูลได้

CLO5

เข้าใจหลักการรักษาความปลอดภัยเว็บไซต์เบื้องต้น

ตารางเรียน 5 วันเต็ม

Day 1: ปูพื้นฐานสู่โลกของ Web

หัวข้อหลัก

  • HTML5 พื้นฐานและโครงสร้างเว็บ (3 ชม.)
  • CSS3 Styling และ Layout เบื้องต้น (3 ชม.)
  • Responsive Design สำหรับมือถือ (2 ชม.)

ทฤษฎี

  • `HTML Tags`, `CSS Selectors`, `Box Model`, `Flexbox`, `Media Queries`

ปฏิบัติ

  • Lab 1.1 - 1.3: สร้างและตกแต่งเว็บแนะนำตัวที่ Responsive
CLO1

Day 2: ทำให้เว็บมีชีวิตชีวา

หัวข้อหลัก

  • JavaScript พื้นฐาน และ DOM Manipulation (4 ชม.)
  • Event Handling และการโต้ตอบ (2 ชม.)
  • Form Validation และ Local Storage (2 ชม.)

ทฤษฎี

  • `Variables`, `Functions`, `DOM Selection`, `Event Listeners`, `Local Storage API`

ปฏิบัติ

  • Lab JavaScript พื้นฐาน
  • Lab 2.1 - 2.3: สร้างเครื่องคิดเลข, เกมทายเลข, และฟอร์มติดต่อ
CLO1

Day 3: ก้าวสู่ Modern Frontend

หัวข้อหลัก

  • รู้จัก `React.js` และ `JSX` (3 ชม.)
  • การสร้าง `Components` และ `Props` (3 ชม.)
  • `State Management` เบื้องต้น (2 ชม.)

ทฤษฎี

  • `React.js Concepts`, `JSX Syntax`, `Props`, `useState Hook`

ปฏิบัติ

  • Lab 3.1 - 3.3: สร้าง `Component` โปรไฟล์, รายการสินค้า, และเครื่องนับคะแนน
CLO1, CLO2

Day 4: สร้างหลังบ้าน (Backend)

หัวข้อหลัก

  • รู้จัก `Node.js` และ `NPM` (2.5 ชม.)
  • `Express.js` สำหรับสร้างเว็บเซิร์ฟเวอร์ (3 ชม.)
  • การทำงานกับ `JSON` และ `API` ง่ายๆ (2.5 ชม.)

ทฤษฎี

  • `Node.js runtime`, `NPM packages`, `Express.js framework`, `HTTP Methods`

ปฏิบัติ

  • Lab 4.1 - 4.3: สร้าง `Server`, `API` ส่งข้อมูล, และรับข้อมูลจาก `Form`
CLO3

Day 5: ประกอบร่าง Full-Stack

หัวข้อหลัก

  • การเชื่อมต่อ `React` กับ `Express API` (3 ชม.)
  • ความปลอดภัยเว็บไซต์เบื้องต้น (2.5 ชม.)
  • ทำโปรเจคสุดท้ายและนำเสนอ (2.5 ชม.)

ทฤษฎี

  • `fetch() API`, `CORS`, `Input Validation`, `HTTPS`

ปฏิบัติ

  • Lab 5.1-5.2: เชื่อม `React App` กับ `API`, เพิ่มความปลอดภัย
CLO2, CLO3, CLO4, CLO5

โปรเจคสุดท้าย: สิ่งที่คุณจะสร้างได้

เว็บไซต์รีวิวร้านอาหาร

นำความรู้ทั้งหมดที่เรียนมาตลอด 5 วัน มาสร้างเว็บแอปพลิเคชันสำหรับรีวิวร้านอาหาร ที่มีทั้งส่วนแสดงผล (`React`) และส่วนจัดการข้อมูล (`Node.js API`) ซึ่งเป็นโปรเจคที่ยอดเยี่ยมสำหรับเริ่มต้น `Portfolio` ของคุณ!

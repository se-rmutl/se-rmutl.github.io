CSS คืออะไร? 🤔

CSS = Cascading Style Sheets

ถ้า HTML คือ โครงสร้าง ของบ้าน, CSS ก็คือ การตกแต่งภายในทั้งหมด

HTML กำหนดว่า:

  • นี่คือหัวข้อใหญ่ (h1)
  • นี่คือย่อหน้า (p)

CSS กำหนดว่า:

  • หัวข้อใหญ่ต้องเป็นสีน้ำเงิน
  • ย่อหน้าต้องมีระยะห่างจากขอบ

CSS ทำให้เว็บไซต์สวยงาม น่าดู และใช้งานง่าย!

ตัวอย่างจริง: HTML vs HTML + CSS

นี่คือโค้ด HTML เดียวกัน แต่แสดงผลต่างกันเมื่อมี CSS เข้ามาช่วย

4/31

3 วิธีการเขียน CSS 📝

1. Inline CSS (ใน HTML Tag)

<h1 style="color: red; font-size: 30px;">หัวข้อสีแดง</h1>

2. Internal CSS (ใน <head>)

<head>
    <style>
        h1 { color: red; font-size: 30px; }
    </style>
</head>

3. External CSS (ไฟล์แยก) ⭐ แนะนำ

<head>
    <link rel="stylesheet" href="style.css">
</head>

ดีที่สุด! เพราะจัดการง่าย, นำกลับมาใช้ซ้ำได้, และทำให้โค้ดสะอาด

External CSS - วิธีที่ดีที่สุด! ✨

1. สร้างไฟล์ style.css

สร้างไฟล์ใหม่และบันทึกด้วยนามสกุล .css

ในไฟล์ index.html เพิ่ม <link> ใน <head>

    <title>My Website</title>
    <link rel="stylesheet" href="style.css">
</head>
6/31

CSS Syntax พื้นฐาน 📚

selector {
    property: value;
}
h1 { color: blue; }
↑ Selector      ↑ Property      ↑ Value
  • Selector = ตัวเลือก (เช่น h1, p, .card)
  • Property = คุณสมบัติ (เช่น color, font-size)
  • Value = ค่า (เช่น blue, 16px)
7/31

CSS Selectors พื้นฐาน 🎯

1. Element Selector

เลือกจากชื่อ Tag

p { color: green; }

2. Class Selector (.)

เลือกจาก class

.highlight { background: yellow; }

3. ID Selector (#)

เลือกจาก id

#header { background: blue; }

Tips: เรานิยมใช้ Class Selector มากที่สุดในการทำงานจริง เพราะมีความยืดหยุ่นสูง

8/31

ความแตกต่าง Class vs ID 🔍

Class (.) - ใช้ซ้ำได้

เหมือน "ป้ายชื่อกลุ่ม" ใช้สำหรับจัดสไตล์ให้ element หลายๆ ตัวที่หน้าตาเหมือนกัน

<button class="btn-primary">OK</button>
<button class="btn-primary">Submit</button>

ID (#) - ห้ามซ้ำ

เหมือน "เลขบัตรประชาชน" ในหนึ่งหน้าจะใช้ชื่อ ID นี้ได้แค่ครั้งเดียวเท่านั้น

<div id="main-navigation">...</div>

กฎทอง: Class = หลายคน, ID = คนเดียว

  • ตัวเลือก D เป็น Inline Style ซึ่งจะเปลี่ยนสีแค่ p Tag เดียว ไม่ใช่ทั้งหมด

Text Properties 📝

การจัดแนว:

text-align: left | center | right | justify;

การตกแต่งข้อความ:

text-decoration: underline | line-through | none;

ระยะห่างบรรทัด:

line-height: 1.5;    /* 1.5 เท่าของฟอนต์ */
line-height: 24px;   /* ความสูงเฉพาะ */

Background Properties 🎨

สีพื้นหลัง:

background-color: lightblue;

รูปพื้นหลัง:

background-image: url('image.jpg');
background-size: cover;
background-repeat: no-repeat;
background-position: center;

ทั้งหมดในบรรทัดเดียว:

background: url('bg.jpg') no-repeat center / cover;

CSS Box Model - หัวใจของ Layout 📦

ทุก HTML Element บนหน้าเว็บคือ "กล่อง" สี่เหลี่ยม ซึ่งประกอบด้วย 4 ชั้น:

Margin
Border
Padding
Content
  • Content: เนื้อหาจริง (ข้อความ, รูปภาพ)
  • Padding: ระยะห่างภายในขอบ
  • Border: เส้นขอบของกล่อง
  • Margin: ระยะห่างภายนอกขอบ

Flexbox Properties รายละเอียด 📋

สำหรับ Container (ตัวห่อหุ้ม)

.flex-container {
  display: flex;
  flex-direction: row | column;
  justify-content: space-between;
  align-items: stretch;
  flex-wrap: wrap;
}

สำหรับ Items (ของข้างใน)

.flex-item {
  flex: 1; /* ขยายเต็มพื้นที่ */
  flex-grow: 2;
  flex-shrink: 0;
}
25/31

ตัวอย่าง Flexbox Layout 💡

HTML:

<div class="header">
    <div class="logo">โลโก้</div>
    <div class="menu">เมนู</div>
    <div class="search">ค้นหา</div>
</div>

CSS:

.header {
    display: flex;
    justify-content: space-between;
    align-items: center;
}
.logo { flex: 1; }
.menu { flex: 2; text-align: center; }
.search { flex: 1; text-align: right; }

Position Properties 📍

ใช้สำหรับกำหนดตำแหน่งของ Element แบบพิเศษ

  • static: (ค่าเริ่มต้น) อยู่ในตำแหน่งปกติตามเอกสาร
  • relative: สามารถเลื่อนจากตำแหน่งเดิมได้ โดยใช้ top, right, bottom, left
  • absolute: หลุดออกจาก Flow ปกติ และยึดตำแหน่งจาก parent ที่เป็น relative ตัวแรก
  • fixed: ติดอยู่กับหน้าจอ แม้จะเลื่อน scroll bar ก็ตาม (เช่น เมนูบนสุด)

การใช้ CSS Classes ในทางปฏิบัติ 🏷️

เราสามารถใช้หลาย Class ใน Element เดียวเพื่อประกอบสไตล์ต่างๆ เข้าด้วยกัน

HTML:

<div class="card shadow">
    <h2 class="card-title">หัวข้อการ์ด</h2>
    <p class="card-text">เนื้อหาในการ์ด</p>
    <button class="btn btn-primary">ปุ่ม</button>
</div>

CSS:

.card { background: white; padding: 20px; }
.shadow { box-shadow: 0 4px 8px rgba(0,0,0,0.1); }
.btn { padding: 10px 20px; }
.btn-primary { background: #007bff; color: white; }

CSS Units หน่วยวัด 📏

หน่วยแน่นอน (Absolute)

  • px (พิกเซล) - ใช้บ่อยที่สุด
  • pt (Point) - สำหรับการพิมพ์
  • cm (เซนติเมตร)

หน่วยสัมพัทธ์ (Relative)

  • % (เปอร์เซ็นต์ของ parent)
  • rem (เท่าของขนาดฟอนต์ root) - **แนะนำสำหรับฟอนต์**
  • vw, vh (เปอร์เซ็นต์ของความกว้าง/สูงหน้าจอ)

CSS Specificity - กฎการชนะแพ้ ⚖️

เมื่อมี CSS หลายตัวเลือก Element เดียวกัน เบราว์เซอร์จะใช้กฎ "ความเฉพาะเจาะจง" เพื่อตัดสินว่าสไตล์ไหนจะถูกใช้งาน

ลำดับความสำคัญ (จากมากไปน้อย):

  1. Inline styles (1000 คะแนน) - style="..."
  2. IDs (100 คะแนน) - #header
  3. Classes (10 คะแนน) - .highlight
  4. Elements (1 คะแนน) - p

กฎทอง: CSS ที่เฉพาะเจาะจงมากกว่าจะชนะ!

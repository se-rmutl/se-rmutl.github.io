Day 1 Blueprint: From Structure to Style

สรุปภาพรวมการสร้างเว็บไซต์ในวันแรก: การวางโครงสร้างด้วย HTML, การตกแต่งด้วย CSS, และการทำให้สวยงามทุกขนาดหน้าจอ

STEP 1: The Structure (HTML)

การวางโครงสร้างเว็บ เหมือนการเขียนแบบแปลนบ้าน

หัวใจสำคัญของ HTML

  • Basic Structure: ทุกหน้าเว็บต้องมีโครงสร้างพื้นฐาน <!DOCTYPE>, <html>, <head>, และ <body>
  • Semantic Tags: ใช้ Tags ที่มีความหมาย (<header>, <main>) เพื่อให้โครงสร้างของเว็บชัดเจนและเป็นมาตรฐาน
  • Attributes: เพิ่มคุณสมบัติให้ Tags เช่น href สำหรับลิงก์, src และ alt สำหรับรูปภาพ

Infographic: Semantic HTML

<body>
  <header>...</header>
  <nav>...</nav>
  <main>
    <section>...</section>
    <article>...</article>
  </main>
  <footer>...</footer>
</body>

STEP 2: The Design (CSS)

การตกแต่งเว็บให้สวยงาม เหมือนการเลือกสีและเฟอร์นิเจอร์

หัวใจสำคัญของ CSS

  • Selectors: เลือกองค์ประกอบที่ต้องการตกแต่ง โดยใช้ชื่อ Tag, .class (สำหรับหลายชิ้น), หรือ #id (สำหรับชิ้นเดียว)
  • Box Model: เข้าใจว่าทุกอย่างคือ "กล่อง" ที่มี Content, Padding (ภายใน), Border (ขอบ), และ Margin (ภายนอก)
  • Flexbox & Grid: เครื่องมือสมัยใหม่สำหรับจัดวาง Layout ให้ยืดหยุ่นและเป็นระเบียบ

STEP 3: The Adaptability (Responsive)

การทำให้เว็บปรับตัวได้ เหมือนการสร้างบ้านที่ขยายขนาดได้

หัวใจสำคัญของ Responsive

  • Viewport Meta Tag: โค้ด <meta name="viewport" ...> ที่ต้องมีใน <head> เพื่อให้เบราว์เซอร์ปรับขนาดเว็บตามอุปกรณ์
  • Mobile-First Approach: เริ่มออกแบบสำหรับหน้าจอเล็ก (มือถือ) ก่อน แล้วค่อยขยาย Layout สำหรับจอที่ใหญ่ขึ้น
  • Media Queries: ใช้ @media เพื่อกำหนดสไตล์ที่แตกต่างกันสำหรับแต่ละขนาดหน้าจอ (Breakpoint)

Infographic: Mobile-First Approach

/* 1. Mobile Styles (Default) */
.container { width: 100%; }

/* 2. Tablet and Up */
@media (min-width: 768px) {
  .container { width: 90%; }
}
